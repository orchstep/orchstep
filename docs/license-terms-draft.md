### Seat Definition
A "seat" is a unique machine on which OrchStep Pro features are activated. OrchStep tracks machine activations locally. Organizations are responsible for ensuring their seat count matches their license.

### Grace Period
If your subscription lapses, Pro features remain available for 14 days with a CLI warning. After 14 days, Pro features are deactivated. Workflow execution (orchstep run) is never affected by license status.
