# orchstep

**YAML-first workflow orchestration engine.**

This package installs the OrchStep binary for your platform.

## Install

```bash
pip install orchstep
```

## Usage

```bash
orchstep run deploy --var env=staging
orchstep lint my-workflow.yml
orchstep list-tasks
```
