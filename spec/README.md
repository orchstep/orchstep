# OrchStep Language Specification

The OrchStep YAML workflow specification defines the format for `orchstep.yml` files.

## Contents

- `workflow-schema.json` — JSON Schema for workflow validation (coming soon)
- `module-schema.json` — JSON Schema for module validation (coming soon)
- `functions/` — Function interface reference (shell, http, git, assert, transform, render, wait)
- `cli-output-schemas/` — JSON Schema for `--format json` output
- `variables.md` — Variable scoping and precedence rules
- `control-flow.md` — Conditionals, loops, and branching
- `error-handling.md` — Try/catch, retry, error modes
- `templates.md` — Go templates and Sprig functions
